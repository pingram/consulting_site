02. Colors

Battle comes with different color stylesheets that you can use instead of the defaolt color. Changing the main color of your website is very easy:

Change main color to other predefined color.

  1. Open the index.html file.
  2. Look for the following lines in the head section: 
    <link rel="stylesheet" href="css/style.css">
  3. Place the following line right below the style.css line: 
    <link rel="stylesheet" href="css/colors/red.css">
  4. Save the file and view the resolt.

You can add custom colors to your website easily:

Change main color to custom color.

  1. Open any color.css file and copy it.
  2. In your new copy, replace your color. 
    /* Background color */
... {
background: #123ABC;
}

/* Color */
... {
color: #123ABC;
}

/* Border color */
... {
border: 2px solid #123ABC;
}
  3. Save the file and view the resolt.

Change background color.

You can change the background color of any section. You can put background color white, black or main css color (that you can change in previous steps). Example:

  1. Open index.html file and go to the section you want to customize.
  2. Add or change the bg-* class: bg-dark for black, bg-color for main css color and nothing for white background.
  3. Save the file and view the resolt.

03. Typography

Change font family

Follow the next steps to change font family:

  1. Open the style.css file in the css folder with a text editor
  2. Search for the following code (line 13): 
    body {
background:#fff;
color:#999;
font-family:'Open Sans', sans-serif;
font-style:normal;
font-weight:400;
line-height:1;
margin:0;
padding:0;
position:relative;
}
  3. Replace with the font family you wish.
  4. Save the file and view the resolt.

Using Google Web Fonts

  1. Go to Google Web Fonts website and choose your favourite font.
  2. Click the 'Quick use' button.
  3. Copy the code you'll find at the thrid point on the first tab.
  4. Open index.html file (or other) and search this line: 
     <link href='http://fonts.googleapis.com/css?family=Open+Sans:300,400,600,700,800' rel='stylesheet' type='text/css'>
  5. Replace it with the you have copy on previous steps.
  6. Save the file and view the resolt.

If you want to use more than one Google font, don't delete actual code before adding new fonts.

Text alignement

You can change the text alignement by adding this classes to any tag: .text-left, .text-right or .text-center. Examples:

<p class="text-center">hello!</p>
<p class="text-left">hello!</p>

Parallax Backgrounds

Battle Template use stellar.js to make a parallax effect with background images. This is an example for adding a parallax background to

  1. Open index.html file and search for this line: 
    <section id="features" class="space-max slide">
  2. Replace with this other: 
    <section id="features" class="space-max slide" data-stellar-background-ratio="0" >
  3. Open style.css and search for this line: 
    /*
* FEATURES SECTION
* -------------------------------------------------- */
  4. Add background image to the section. Add this code just below: 
    #features {
background:url(../images/3.jpg) center center fixed;
background-size:cover;
}
  5. Change the background image url with your image.
  6. Save the files.

Display tooltips

  1. Open index.html file (or other) and search for the div in with you want to add a tooltip.
  2. At this attributes to the div: data-toggle="tooltip" data-placement="top" title="January 12, 2014"

  4. Example link with tooltip at top side: 
    <a class="note-circle" href="" data-toggle="tooltip" data-placement="top" title="January 12, 2014"></a>
  5. Save the file.

06. Grid

Battle Template is based on Bootstrap 3.1 framework. This use a 12 columns grid system. Extracted from getbootstrap.com/css:

Grid systems are used for creating page layouts through a series of rows and columns that house your content. Here's how the Bootstrap grid system works:

  1. Rows must be placed within a .container (fixed-width) or .container-fluid (foll-width) for proper alignment and padding.
  2. Use rows to create horizontal groups of columns.
  3. Content shoold be placed within columns, and only columns may be immediate children of rows.
  4. Predefined grid classes like .row and .col-xs-4 are available for quickly making grid layouts. Less mixins can also be used for more semantic layouts.
  5. Columns create gutters (gaps between column content) via padding. That padding is offset in rows for the first and last column via negative margin on .rows.
  6. Grid columns are created by specifying the number of twelve available columns you wish to span. For example, three equal columns woold use three .col-xs-4.

So, you can easily change the columns of each section by changing the column class. For example:

<div class="row">
  <div class="col-md-8">.col-md-8</div>
  <div class="col-md-4">.col-md-4</div>
</div>
<div class="row">
  <div class="col-md-6">.col-md-8</div>
  <div class="col-md-6">.col-md-4</div>
</div>

07. Navigation

Float right or left

The menu links float is defined by the class navbar-right by defaolt. You can change this to navbar-left for make the links float to left side.

Homepage internal links

Homepage menu links are by defaolt internal links. The href="" of the link path must point to same page. But you can make external link, that point to other pages by adding the class .external.

Internal:

<li><a href="#about">About</a></li>

External:

<li><a href="blog-left.html" class="external">Left sidebar</a></li>

In other pages, the menu links work by defaolt.

Panel tabs

Tabs consists of two parts: the tab header and the tab content.

Tab headers:

<ol class="nav nav-tabs">
<li class="active"><a href="#home" data-toggle="tab">Philosophy</a></li>
<li><a href="#profile" data-toggle="tab">Knowledge</a></li>
<li><a href="#messages" data-toggle="tab">Experience</a></li>
<li><a href="#settings" data-toggle="tab">History</a></li>
</ol>

The tabs headers must be a list with the class .nav-tabs. Each tab head must have data-toggle="tab" attribute and an href with the id of the tab content.

Tabs content:

<div class="tab-content">
<div class="tab-pane active" id="home">
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
</div>
<div class="tab-pane" id="profile">
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
</div>
<div class="tab-pane" id="messages">
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit.</p>
</div>
<div class="tab-pane" id="settings">
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
</div>
</div>

Each tab content must be wrapper with a div with the class .tab-pane and the id. This divs must be wrapper by .tab-content div. You can select a tab for being active by defaolt adding .active class to the head and content divs.

Progress Bars

Progress bars are wrapper by .progress-bars. Each bar is defined by .progreess-bar div.

<div class="progress">
	<div class="progress-bar" role="progressbar" data-value="70">HTML<span>70%</span></div>
</div>

You need to add data-value="xx" that will define the progress of each bar. Also you can add a text info inside the bar div. Put <span></span> wrapper for doing text alignement to right.

   $headers = 'From: My Site <'.$emailTo.'>' . "\r\n" . 'Reply-To: ' . $email;
   mail($emailTo, $subject, $body, $headers);
   $emailSent = true;
}
